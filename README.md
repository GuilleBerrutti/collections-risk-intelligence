# Collections Strategy Analytics 🚀

Este proyecto presenta una solución integral de **Business Intelligence** diseñada para optimizar la recuperación de activos en una Fintech. El dashboard permite a los gestores de riesgo identificar cuellos de botella en la operación y priorizar la contactabilidad basada en datos.

## 📊 Insights Clave del Proyecto

* **Segmentación de Riesgo:** Clasificación de deudores por rangos de ingresos, identificando que la mayor concentración de deuda se encuentra en el segmento de ingresos medios.

* **Eficacia de Canales:** Comparativa de rendimiento entre WhatsApp y Llamadas para maximizar la tasa de compromiso de pago.

* **Priorización Operativa:** Tabla dinámica con el Top 10 de deudores críticos para acción inmediata.

## 🛠️ Stack Tecnológico

* **Power BI Desktop:** Modelado de datos y visualizaciones avanzadas.

* **DAX (Data Analysis Expressions):** Implementación de lógica de negocios compleja, incluyendo rankings dinámicos y medidas de eficacia corregidas.

* **ETL:** Procesamiento y limpieza de datos desde fuentes CSV/Excel.

## 📁 Estructura del Repositorio
* `/Data`: Dataset anonimizado utilizado para el análisis.
* `/Dashboard`: Archivo `.pbix` con el reporte interactivo.
* `/Screenshots`: Capturas de pantalla del dashboard en funcionamiento.
